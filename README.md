# Sales Advertising Linear Regression Analysis

## Project Overview
This project analyzes the relationship between advertising spending and sales using a Linear Regression model.

## Objective
To understand how advertising budget influences sales and build a predictive model.

## Technologies Used
Python  
Pandas  
Matplotlib  
Scikit-learn  

## Analysis Steps
1. Data loading and preprocessing
2. Correlation analysis
3. Train-test split
4. Linear regression modeling
5. Model evaluation using RMSE and R²
6. Residual analysis
7. Data visualization

## Model Results

Regression Equation:

Sales = 1.915 × Advertising − 17.92

The model shows a positive relationship between advertising budget and sales.

## Example Prediction

The model can estimate expected sales for a given advertising budget.
